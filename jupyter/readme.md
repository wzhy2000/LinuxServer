如何正确关闭Jupyter
- Jupyter页面右上角的quit，点击
- 在服务器命令行，`python -m jupyter notebook stop {port}`
  ![image](pics/04.png)
- 远程命令，`ssh {username}@{ip} "pkill -u {username} jupyter"`
- 管理员远程命令，`ssh {username}@{ip} "jupyter notebook stop {port}"`
